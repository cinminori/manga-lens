Features

功能特色

為漫畫翻譯場景量身打造的使用體驗。

📷

框選即翻譯

拖曳選取漫畫區域，自動截圖、OCR 辨識、翻譯一氣呵成。

🔄

連續框選模式

翻譯完一格後繼續框選下一格，不需重複啟動，整頁翻譯更流暢。

👆

結果框互動

拖曳移位、長按透明對照原文、一鍵重翻或複製翻譯文字。

📱

手機也能用

支援 Android 觸控操作，在 Kiwi Browser 等支援擴充套件的瀏覽器上使用。

🔧

Install

安裝方式

支援桌面版 Chrome 與 Android 手機瀏覽器。

🖥️ 桌面版 Chrome

  1. 開啟 chrome://extensions/
  2. 右上角開啟「開發人員模式」
  3. 點擊「載入未封裝項目」
  4. 選擇擴充套件資料夾
  5. 完成！網頁右下角會出現浮動工具列

📱 Android 手機

  1. 安裝支援擴充套件的瀏覽器（如 Kiwi Browser）
  2. 將擴充套件資料夾打包成 .zip
  3. 在瀏覽器擴充功能頁面匯入 zip 檔
  4. 完成！用手指觸控操作框選
